## テスト失敗とその対応

### 初回実行時のエラー

**エラー1**: `notify_procast_data_missing()`に`hour`引数が不足
- **原因**: メソッドシグネチャが`notify_procast_data_missing(self, hour: int)`で、`hour`引数が必須
- **修正**: テストコードで`hour=20`を引数に追加
- **結果**: 修正後、テスト成功

**エラー2**: スケジューラーの`shutdown()`エラー
- **原因**: スケジューラーが`start()`されていない状態で`shutdown()`を呼び出した
- **修正**: テストコードから`scheduler.shutdown(wait=False)`を削除
- **結果**: 修正後、テスト成功
